## Features

- **Premium UI** — Apple/Linear/Stripe-level polish with glassmorphism, spring animations, and micro-interactions
- **Floating Pill Navigation** — Bottom nav with icon-only tabs that expand to icon+text on active/hover
- **Smart Scroll Behavior** — Nav shrinks to dots on scroll down, reveals on scroll up/stop
- **Skeleton Loader** — Full-screen loading with Hindi shimmer animation
- **Animated Hero** — Word-by-word reveal with gradient text animation
- **WhatsApp Integration** — Direct booking via WhatsApp with pre-filled form data
- **Responsive Design** — Works seamlessly on desktop, tablet, and mobile
- **Dark Footer** — Developer signature with GitHub and Instagram links
